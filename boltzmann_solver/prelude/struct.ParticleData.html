Expand description

Particle type

The particle type is only very basic, containing only the minimum information required for cosmological purposes. Specifically, it only keeps track of the particle’s spin, mass and whether it is complex. It does not keep track of transformation properties and interactions.

By default, the degrees of freedom is calculated based on the particle’s spin and whether it is complex or not. An arbitrary number can be specified which will override this calculation (which can be used for multiplets for example).

For hashing purposes, only the particle’s name is taken into account.

In the long run, it is intended that this type be replaced by another one.

By default, this is assumed to be false and can be set to true with [Particle::own_antiparticle].

Specify how many internal degrees of freedom this particle has. These can be because the particle is complex or any other internal gauge symmetries.

This is a multiplicative factor to the degrees of freedom. For a ‘pseudo’ particle such as $B-L$, this should be set to zero.

pub fn degrees_of_freedom(&self) -> f64

Return the number of degrees of freedom for the underlying particle.

The general formula is:

N_\text{int} \times N_\text{spin}

where $N_\text{int}$ are the internal degrees of freedom, $N_\text{spin}$ are the spin degrees of freedom.

The spin degrees of freedom are:

  • spin-0: 1
  • spin-1/2: 2
  • spin-1: 2 for massless, 3 for massive
  • spin-3/2: 4
  • spin-2: 2 for massless, 5 for massive

or more generally, for half-integer spins n/2, the degrees of freedom is n + 1; and for integer spin n > 0, the degrees of freedom are 2 for massless or 2n + 1 for massive.

Panics

Panics if a particle with spin greater than 2 is given.

pub fn propagator<M: Model>(&self, model: &M, s: f64) -> Complex<f64>

Return the propagator denominator for the particle.

The propagator is defined as follow:

\mathcal{P}_i^{-1}(s) = s - m_i^2 + i \theta(s) m_i \Gamma_i,

where $\theta$ is the Heaviside step function.

source

pub fn ris_propagator<M: Model>(&self, model: &M, s: f64) -> SinglePropagator

Return the RIS propagator for the particle.

The RIS propagator is defined such that a real intermediate state is substracted from the propagator. This is done to avoid a double-counting issue of with $2 \xleftrightarrow{P} 2$ interaction with some intermediate particle $P$ and the combination of the two 3-particle interactions 2 ↔ P and P ↔ 2.

The definition is follows that of a regular propagator except in the following case:

\mathcal{P}_i(s) \mathcal{P}_i^*(s)
= \frac{(s - m_i^2)^2 - (m_i \Gamma_i)^2}{[(s - m_i^2)^2 + (m_i \Gamma_i)^2]^2}.
