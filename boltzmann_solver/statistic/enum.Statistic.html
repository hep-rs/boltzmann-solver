pub enum Statistic {
    FermiDirac,
    BoseEinstein,
    MaxwellBoltzmann,
    MaxwellJuttner,
}
Expand description

The statistics which describe the distribution of particles over energy states. Both Fermi–Dirac and Bose–Einstein quantum statistics are implemented, as well as the classical Maxwell–Boltzmann statistic.

Variants

FermiDirac

Fermi–Dirac statistic describing half-integer-spin particles:

f_{\text{FD}} = \frac{1}{\exp[(E - \mu) \beta] + 1}.
BoseEinstein

Bose–Einstein statistic describing integer-spin particles:

f_{\text{BE}} = \frac{1}{\exp[(E - \mu) \beta] - 1}.
MaxwellBoltzmann

Maxwell–Boltzmann statistic describing classical particles:

f_{\text{MB}} = \exp[-(E - \mu) \beta].
MaxwellJuttner

Maxwell–Jüttner statistic describing relativistic classical particles:

f_{\text{MJ}} = \frac{E \beta \sqrt{E^2 - m^2}}{m K_2(m \beta)} \exp[- E \beta].

fn number_density(&self, mass: f64, mu: f64, beta: f64) -> f64

Return number density for a particle following the specified statistic.

n = \frac{1}{2 \pi^2} \int_{m}^{\infty} f_{i} u \sqrt{u^2 - m^2} \dd u

Implementation Details

Both the Fermi–Dirac and Bose–Einstein statistic rely on numerical integration and thus are fairly slow and are prone to errors in certain regimes. The Maxwell–Boltzmann and Maxwell–Jüttner distributions offer exact implementations.

Return number density for a particle following the specified statistic, normalized to the number density of a massless boson with a single degree of freedom.

fn massless_number_density(&self, mu: f64, beta: f64) -> f64

Return number density for a massless particle following the specified statistic.

n = \frac{1}{2 \pi^2} \int_{0}^{\infty} f_{i} u^2 \dd u

Implementation Details

All four statistics have exact implementations and do not rely on any numerical integration.
