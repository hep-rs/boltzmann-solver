Particle type

The particle type is only very basic, containing only the minimum information required for cosmological purposes. Specifically, it only keeps track of the particle's spin, mass and whether it is complex. It does not keep track of transformation properties and interactions.

By default, the degrees of freedom is calculated based on the particle's spin and whether it is complex or not. An arbitrary number can be specified which will override this calculation (which can be used for multiplets for example).

In the long run, it is intended that this type be replaced by another one.

Fields

spin: u8

The spin is stored as twice the spin, so a spin-½ particle has spin == 1 and a spin-1 particle has spin == 2

mass: f64

Mass of the particle in GeV
